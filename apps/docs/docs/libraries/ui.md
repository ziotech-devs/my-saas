# UI Components

Shared component library at `libs/ui` based on [Shadcn/UI](https://ui.shadcn.com/) and Radix UI primitives.

## Usage

```tsx
import { Button } from "@my-saas/ui";
```

All components are styled with Tailwind and support `class-variance-authority` for variants.
