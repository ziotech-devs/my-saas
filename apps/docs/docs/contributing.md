# Contributing

See [CONTRIBUTING.md](https://github.com/ziotech-devs/my-saas/blob/main/CONTRIBUTING.md) in the root of the repository.

## Code Style

- Functional React components, named exports only
- Types over interfaces
- String literals over enums
- Tailwind for all styling
- kebab-case file names

See `CLAUDE.md` at the repo root for the full style guide.
